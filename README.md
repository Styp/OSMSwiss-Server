# OSMSwiss-Server

## What is this about?
OSMSwiss-Server is a box to develop, style and create beautiful maps in TileMill2

## Prerequisites
* Linux OS / Windows
* Vagrant (https://www.vagrantup.com/)
* TillMill2 (https://www.mapbox.com/tilemill/)
